Vos Encaissements Simplifiés

Options de Réception des Ventes

Choisissez la méthode qui vous convient le mieux pour collecter les fonds de vos ventes de tickets Mobile Money.

Via Votre Compte FedaPay Personnel

Reliez vos numéros Mobile Money à un compte FedaPay personnel. Les fonds de vos ventes sont crédités directement sur ce compte.

Frais FedaPay : 1.8% par transaction. Vous contrôlez qui supporte ces frais (vous ou le client).

Via Le Portefeuille Intégré EVISIONS

Utilisez le portefeuille intégré au système pour encaisser vos ventes si vous ne souhaitez pas créer de compte FedaPay personnel.

Commission EVISIONS : 10% sur les transactions encaissées via ce portefeuille.
